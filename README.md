# Haseeb Gul – Portfolio Website


A responsive personal portfolio website built using HTML5, CSS3, and JavaScript.
This project showcases my skills, education, work experience, and projects with a clean, modern, and user-friendly design.


## Live Preview


[Live Demo](https://haseeb-portfolio-site.netlify.app/)


## Features


-Fully responsive layout for desktops, tablets, and mobiles

-Home section with introduction and social links

-About section with personal bio and profile image

-Education timeline with icons and dates

-Skills section with animated progress bars

-Work/projects section with clickable project cards

-Contact form integrated with Netlify Forms

-Footer with social media links

-Smooth scroll and scroll reveal animations


## Technologies Used


-HTML5

-CSS3

-JavaScript (ScrollReveal animations)

-Font Awesome icons

-Netlify Forms (for contact submissions)
